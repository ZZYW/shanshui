Thank you for choosing Iridescence Thin Film !
This is a shader package which offers high quality iridescence thin film realtime rendering in unity3d.

Specification:
  => "Iridescence Thin Film/Diffuse" render iridescence thin film without bump mapping.
  => "Iridescence Thin Film/Bumped Diffuse" render iridescence thin film with bump mapping.
  => "IridescenceThinFilm.cs" is responsible for send appropriate material parameters to shader.
The demo scene demonstrates all features, please refer to it for the usage example.
